# 📊 Интерактивный тренажер линейной регрессии (Streamlit Web App)

Веб-приложение для интерактивного изучения линейной регрессии (метод наименьших квадратов), оценки качества моделей ($R^2$, MSE) и глубокого анализа остатков.

🔗 **[ПОСМОТРЕТЬ ПРИЛОЖЕНИЕ В ЖИВУЮ] (https://interactive-linear-regression-ggjrrbsylgfnvvgj4bbdya.streamlit.app/)**

---

## 🎯 Особенности проекта
- **Математический движок на NumPy:** Алгоритм линейной регрессии (МНК/OLS) написан вручную через матричные вычисления (нормальное уравнение) без использования `scikit-learn`.
- **Генерация синтетических данных:** Возможность интерактивно управлять объемом выборки, уровнем случайного шума и формой тренда (линейный/квадратичный).
- **Диагностика моделей (Анализ остатков):** Реализованы три графика для проверки условий Гаусса-Маркова: Scatter plot остатков, гистограмма распределения и график квантиль-квантиль (Q-Q Plot).

## 🛠️ Стек технологий
- **Python**
- **Streamlit** (построение интерактивного веб-интерфейса и реактивное управление состоянием через `session_state`)
- **NumPy** (матричные операции, генерация распределений)
- **Matplotlib & Seaborn** (визуализация регрессии и распределения остатков)
- **SciPy** (построение вероятностного графика Q-Q Plot)

## 📌 Как это работает (Бизнес-ценность)
Проект служит отличным демонстрационным и обучающим инструментом:
1. При **Линейном тренде** остатки распределены нормально, а модель показывает высокую точность ($R^2 \approx 1$).
2. При переключении на **Квадратичный тренд** приложение наглядно визуализирует проблему **недообучения (Underfitting)**: на графике остатков мгновенно прослеживается четкая параболическая зависимость, сигнализирующая о нелинейности реальных данных.

## 🏃‍♂️ Локальный запуск
Если вы хотите запустить проект у себя на компьютере:
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Запустите Streamlit-сервер:
   ```bash
   streamlit run app.py
   ```
